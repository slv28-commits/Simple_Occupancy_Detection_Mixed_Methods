# Simple_Occupancy_Detection_Mixed_Methods
Here is an updated version of my last occupancy detection program, this time using adaptive thresholding and comparison to a reference with white squares, and mean, min, and max comparison to a reference with black squares. 
This method has higher accuracy than my previous attempt, however, it does need to be tested more.

Please note that the main reason I used different methods for black and white squares was that white squares had a very visible wooden grain that messed up simple detection.
